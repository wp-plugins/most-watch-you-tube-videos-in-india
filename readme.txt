This section describes how to install the plugin and get it working.

e.g.

1. Upload 'most_watched_videos_in_india_today' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Widgets and add this plug in in the sidebar and you are all set.
